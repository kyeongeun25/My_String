Web Service
Web Service는 회사 HomePage처럼 디자인 측면에서
HTML, CSS, JS 등을 활용하여 view page를 만드는 것
이런 Web Service를 제공하기 위해 사용한 컴퓨터, OS 등을
Web Server라고 한다.

Web Application Service
Database와 연동되서 CRUD를 구현하고, 
데이터의 추가, 검색, 수정, 삭제 등의 기본 업무에 여러가지 연산작업(데이터 가공)을 수행하는데

1.
Web Browser로 부터 데이터를 입력하고 그 데이터를 서버로 전송하여
DataBase에 Insert하고

2.
저장된 DataBase로 부터 데이터를 인출(Select)하여 필요한 정보로 가공 연산하여 
Web Browser를 통해 보여주고 

3. 필요에 따라서 이미 저장된 데이터를 수정하고

4. 활용이 끝나서 불필요한 data는 삭제하는 일련의 여러가지 작업(프로세스)를 수행한다.

** 10/17
form을 만들고 form에 입력된 데이터를 서버로 전송하고, DataBase에 저장하고
검색, 수정, 삭제하는 과정을 작성

WAS(Web Application Service, Server)를 사용한 
어플리케이션은 HTML과 CSS를 활용한 디자인 측면과 
서버로 전송하기 전 JS를 활용한 데이터 검증이 많은 부분을 차지한다.

** HTML과 CSS BootStrap과 W3.css를 일부 활용
** 데이터 검증 부분은 Jquery를 활용하여 복습
